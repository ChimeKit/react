# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-13

### Added

- Initial release
- `ChimeKitProvider` for context and authentication
- `Inbox` component with popover, modal, and drawer variants
- `Bell` component for notification indicator
- `Feed` component for message list display
- `Preferences` and `PreferencesDialog` components
- `MessageDetails` and `MessageDetailsDialog` components
- `useChimeKit` hook for accessing the client
- CSS variables for theming
- Dark mode support
- HTML sanitization for message content
- TypeScript type definitions
